## 🚀 JavaScript框架

### 主流前端框架
- **[React](https://reactjs.org)** - 用于构建用户界面的JavaScript库
  - **特色**: 组件化、虚拟DOM、丰富生态系统
  - **适用**: 复杂单页应用、企业级项目
- **[Vue.js](https://vuejs.org)** - 渐进式JavaScript框架
  - **特色**: 易学易用、灵活渐进、优秀的开发体验
  - **适用**: 中小型项目、快速原型开发
- **[Angular](https://angular.io)** - 企业级应用框架
  - **特色**: TypeScript集成、依赖注入、完整的工具链
  - **适用**: 大型企业应用、复杂业务逻辑
- **[Svelte](https://svelte.dev)** - 编译时优化的前端框架
  - **特色**: 零运行时开销、小体积、出色性能
  - **适用**: 性能敏感应用、轻量级项目

### Meta框架
- **[Next.js](https://nextjs.org)** - React的生产级框架
  - **特色**: SSR/SSG、API路由、图像优化、自动代码分割
- **[Nuxt](https://nuxt.com)** - Vue.js的全栈框架
  - **特色**: 服务端渲染、自动路由、模块化架构
- **[SvelteKit](https://kit.svelte.dev)** - Svelte的应用框架
  - **特色**: 全栈能力、适配器系统、优秀性能
- **[Astro](https://astro.build)** - 内容驱动的网站框架
  - **特色**: 岛屿架构、多框架支持、零JS默认

## 💄 CSS框架

### 实用优先框架
- **[Tailwind CSS](https://tailwindcss.com)** - 实用优先的CSS框架
  - **特色**: 高度可定制、设计系统友好、零运行时
  - **适用**: 定制化设计、快速原型开发
- **[Tachyons](https://tachyons.io)** - 函数式CSS框架
  - **特色**: 原子化类名、极轻量级、高性能
- **[Winduum](https://winduum.dev)** - 现代CSS组件库
  - **特色**: 基于Web标准、TailwindCSS扩展、渐进式

### 组件化框架
- **[Bootstrap](https://getbootstrap.com)** - 最流行的CSS框架
  - **特色**: 丰富组件库、响应式网格、稳定可靠
  - **适用**: 快速开发、企业项目、原型制作
- **[Bulma](https://bulma.io)** - 现代Flexbox CSS框架
  - **特色**: 纯CSS、无JavaScript依赖、模块化
- **[Foundation](https://get.foundation)** - 企业级响应式框架
  - **特色**: 可访问性、移动优先、灵活网格

### 极简框架
- **[Pico CSS](https://picocss.com)** - 语义化HTML的极简框架
  - **特色**: 类名极少、语义化、零配置
- **[Pure CSS](https://purecss.io)** - 轻量级CSS模块集合
  - **特色**: 模块化、极轻量、Yahoo出品
- **[Skeleton](https://getskeleton.com)** - 简单的响应式样板
  - **特色**: 极简设计、快速原型、400行CSS

### 特色框架
- **[Materialize](https://materializecss.com)** - Material Design CSS框架
- **[UIkit](https://getuikit.com)** - 轻量级模块化前端框架
- **[Semantic UI](https://semantic-ui.com)** - 人性化HTML界面框架
- **[Yumma CSS](https://www.yummacss.com)** - 缩写式实用类CSS框架
- **[Toucaan](https://www.toucaan.com)** - 内在设计CSS框架

### React/Vue组件库
- **[Mantine](https://mantine.dev)** - 功能丰富的React组件库
  - **特色**: 120+组件、70+Hooks、优秀的开发体验
- **[Chakra UI](https://chakra-ui.com)** - 简单、模块化、可访问的React组件库
- **[Ant Design](https://ant.design)** - 企业级React UI设计语言
- **[Element Plus](https://element-plus.org)** - Vue 3组件库
- **[Vuetify](https://vuetifyjs.com)** - Vue.js Material Design组件框架

## 🏗️ 构建工具和打包器

### 现代构建工具
- **[Vite](https://vitejs.dev)** - 下一代前端构建工具
  - **特色**: 极速热重载、ES模块原生支持、优化的构建输出
- **[Turbopack](https://turbo.build/pack)** - Webpack的Rust继任者
  - **特色**: 极速构建、增量计算、Next.js内置
- **[esbuild](https://esbuild.github.io)** - 极速JavaScript打包器
  - **特色**: Go编写、极高性能、简单配置

### 传统打包器
- **[Webpack](https://webpack.js.org)** - 模块打包器
  - **特色**: 丰富的插件生态、灵活配置、成熟稳定
- **[Rollup](https://rollupjs.org)** - JavaScript模块打包器
  - **特色**: 树摇优化、ES模块原生、适合库开发
- **[Parcel](https://parceljs.org)** - 零配置Web应用打包器
  - **特色**: 开箱即用、自动优化、多核处理

### 开发服务器
- **[Bun](https://bun.sh)** - 极速JavaScript运行时和包管理器
- **[Snowpack](https://www.snowpack.dev)** - 更快的Web开发构建工具
- **[WMR](https://wmr.dev)** - 微小的Web开发工具
